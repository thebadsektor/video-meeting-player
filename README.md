# MeetingViewer

A Node.js + Express + Vite project for serving video meeting viewers with API routes.  

This project uses TypeScript and serves both API endpoints and a front-end client.  

---

## Prerequisites

- Node.js v20+ (recommended LTS)  
- npm 9+  

---

## Setup

1. Clone the repository:

```bash
git clone <your-repo-url>
cd MeetingViewer
```
2.	Install dependencies:
```
npm install
```
